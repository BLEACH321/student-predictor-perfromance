## 📌 Prompt: Project Guidelines

### Student Performance Predictor (ML + Cloud)

Act as a Senior Project Manager responsible for ensuring quality, scalability, and successful delivery of the project.

Provide detailed guidelines for executing the "Student Performance Predictor" project with the following focus areas:

1. Technical Guidelines:

   * Start with a simple machine learning model (Linear Regression)
   * Ensure proper data preprocessing and validation
   * Maintain modular architecture

2. Coding Guidelines:

   * Use clean and readable code
   * Follow proper file structure
   * Add comments and documentation

3. Cloud Guidelines:

   * Use Firebase for storing student data
   * Secure API keys and database rules
   * Ensure proper data handling practices

4. UI/UX Guidelines:

   * Keep interface simple and user-friendly
   * Minimize input complexity
   * Display results clearly

5. Performance Guidelines:

   * Ensure fast prediction response
   * Optimize backend processing
   * Avoid unnecessary computations

6. Testing Guidelines:

   * Test system with multiple inputs
   * Validate model accuracy
   * Debug errors before deployment

7. Documentation Guidelines:

   * Maintain proper project report
   * Include architecture diagrams
   * Prepare explanation for viva

8. Presentation Guidelines:

   * Clearly explain project problem and solution
   * Demonstrate working model
   * Be ready to explain:

     * Model selection
     * Cloud integration

9. Risk Management:

   * Handle low accuracy by improving model
   * Improve dataset quality if needed
   * Test cloud integration thoroughly

Ensure all guidelines are:

* Easy to follow
* Practical for students
* Focused on successful project completion
